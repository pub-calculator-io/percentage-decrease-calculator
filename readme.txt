[https://www.calculator.io/percentage-decrease-calculator/](https://www.calculator.io/percentage-decrease-calculator/)

== Usage ==

Add the Percentage Decrease Calculator shortcode to your page, post or sidebar:

`[calcio_percentage_decrease_calculator]`

Add the following code to your template where you would like the Percentage Decrease Calculator to appear:

`<?php display_calcio_percentage_decrease_calculator(); ?>`
